# AI INTERN 

AI INTERN is a multi-page AI-assisted internship guidance portal developed during the Smart India Hackathon (SIH).

## About the Project
This platform helps students discover internships, explore categories, and get skill-based recommendations based on their interests and goals.

It is especially designed for first-generation learners and students from rural areas to simplify access to career guidance and opportunities.

## Features
- Internship category browsing  
- Skill-based internship finder  
- AI-assisted skill pathway guidance  
- Success stories section  
- Contact interface  

## Built With
- HTML  
- CSS  
- AI-assisted tools (for guidance and form integration)
